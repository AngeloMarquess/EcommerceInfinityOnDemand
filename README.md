## 🚀 Sobre o Projeto

O projeto **Infinity on Demand** é uma interface desenvolvida com foco total em **UX/UI e responsividade**. Utilizando um design pautado em tendências modernas como o *Glassmorphism* (efeito de vidro), animações fluidas e layouts flexíveis, o projeto entrega uma experiência de tirar o fôlego para o usuário. 

Além disso, a página conta com um sistema de **captura de leads simulada** que armazena os dados no `localStorage` do navegador e redireciona o usuário perfeitamente para um calendário de agendamento (Calendly), garantindo que nenhum contato seja perdido.

## ✨ Principais Funcionalidades

- **Design Premium & Responsivo:** Interface totalmente adaptável para qualquer dispositivo (Desktop, Tablet e Mobile) evitando quebras de layout.
- **Glassmorphism & Animações:** Efeitos visuais modernos que passam uma sensação de "premium" e profissionalismo.
- **Formulário com Validação & Máscaras:** Campos de contato para Whatsapp com máscara interativa simulando o padrão local `(XX) XXXXX-XXXX`.
- **Navegação Dinâmica (Smooth Scroll):** Barra de navegação transparente que se adapta quando o usuário utiliza o scroll, otimizando o espaço da tela.
- **Pronto para Produção:** Arquitetura limpa utilizando o Vite, otimizada para implantações rápidas em plataformas como *Hostinger*, *Vercel* ou *Netlify*.

## 🛠️ Tecnologias Utilizadas

- **[React.js](https://reactjs.org/)** - Biblioteca JavaScript para construção da interface.
- **[TypeScript](https://www.typescriptlang.org/)** - Superset do JavaScript que adiciona tipagem estática e segurança ao código.
- **[Vite](https://vitejs.dev/)** - Ferramenta de build extremamente rápida.
- **[Lucide React](https://lucide.dev/)** - Biblioteca de ícones elegantes e consistentes.
- **[React Router Dom](https://reactrouter.com/)** - Para lidar com o roteamento interno das páginas (como a de Políticas de Privacidade).

## ⚙️ Como Executar Localmente

Siga os passos abaixo para rodar o projeto no seu ambiente local:

1. Clone o repositório:
```bash
git clone git@github.com:AngeloMarquess/EcommerceInfinityOnDemand.git
```

2. Acesse a pasta do projeto:
```bash
cd EcommerceInfinityOnDemand
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. O servidor será iniciado localmente. Basta abrir o seu navegador no link fornecido no terminal (geralmente `http://localhost:5173/`).
